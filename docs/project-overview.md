# Project Overview

## Title
Diagnosis of a Patient to Detect COVID-19 Using Deep Learning

## Description
This project proposes an AI-assisted diagnosis-support platform for COVID-19. It combines public COVID-19 trend data, patient clinical information, and medical imaging to support faster diagnosis and more informed analysis. The broader motivation is to use deep learning to improve prediction, reporting, and image-based diagnosis during a public-health emergency.

The report description explains that the system was designed to:

- process structured and unstructured COVID-19-related data
- assist image-based diagnosis using CT scans and chest X-rays
- provide user and admin workflows through a web application
- improve prediction and decision support using ANN, CNN, and LSTM models

## Functional Areas

### 1. User Module
- user registration
- login and session handling
- access to current COVID-19 information
- access to clinical and prediction outputs

### 2. Admin Module
- login and user activation
- model-control workflow
- CT scan testing support

### 3. Clinical Process Module
- analysis of symptoms, vitals, and risk factors
- CSV-driven reporting workflow
- support for physicians and researchers

### 4. Imaging and Prediction Module
- CNN-based model training for COVID-19 / normal scan classification
- LSTM-based forecasting on time-series case data
- ANN / deep-learning support for prediction and diagnosis

## Algorithms Mentioned In The Project Material
- Artificial Neural Network (ANN)
- Long Short-Term Memory (LSTM)
- Convolutional Neural Network (CNN)
- Backpropagation (listed as part of the existing-system discussion)
